## О проекте

Современная образовательная среда характеризуется высокой степенью фрагментации информационных ресурсов. Данный бэкенд решает задачу создания унифицированных информационных порталов подразделений. Система построена по архитектуре клиент-сервер и предоставляет надежный REST API для управления пользователями, контентом и структурой сайта.

Основная область применения — цифровая поддержка учебного процесса. Серверная часть обеспечивает разграничение прав доступа для трех ролей (Посетитель, Редактор, Администратор) и управляет всеми данными, включая текстовый и мультимедийный контент.

## Технологический стек

- **Язык:** Python 3.10+
- **Фреймворк:** Django 4.2+
- **REST API:** Django REST Framework (DRF) 3.14+
- **Аутентификация:** JWT (djangorestframework-simplejwt)
- **Контейнеризация:** Docker, docker-compose

## Функциональные возможности (API)

API реализует полную бизнес-логику системы, описанную в требованиях к разработке.

**Роль «Посетитель» (неавторизованный):**
- Получение главной страницы и структуры сайта.
- Просмотр опубликованных страниц и встроенного мультимедийного контента.
- Навигация по меню.

**Роль «Редактор»:**
- Авторизация в системе.
- Получение списка закрепленных за редактором страниц/блоков.
- Изменение текстового содержимого существующих блоков.

**Роль «Администратор»:**
- Конструирование страниц через добавление, удаление и изменение порядка типовых блоков.
- Настройка параметров каждого блока.
- Управление пользователями и назначение прав доступа (ролей) на страницы и блоки.
